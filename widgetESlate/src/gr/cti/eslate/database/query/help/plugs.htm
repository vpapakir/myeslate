Σύνδεσμοι 

Αρχικά, η ψηφίδα εμφανίζει ένα διαθέσιμο σύνδεσμο:

ΕΙΔΟΣ ΣΥΝΔΕΣΜΟΥ

ΣΚΟΠΟΣ

Είσοδος ενός ή περισσότερων πινάκων από μία βάση δεδομένων

Μόλις συνδεθεί η ψηφίδα με μία βάση δεδομένων, εμφανίζει ένα σύνδεσμο για κάθε πίνακα της ΒΔ, της μορφής:

ΕΙΔΟΣ ΣΥΝΔΕΣΜΟΥ

ΣΚΟΠΟΣ

Έξοδος της απάντησης της ερώτησης άλλες ψηφίδες ικανές να εμφανίσουν δεδομένα σε μορφή πίνακα
