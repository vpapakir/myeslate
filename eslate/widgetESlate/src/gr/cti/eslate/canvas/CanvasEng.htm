Plugs

  • Colors: Connect this plug to a colour output plug to get the pen, fill and rubber colours from it.

  • Turtle Commands: Connect this protocol plug to a Turtle component to host it in a new auto-created Turtle page (unless the page visible on top is already a turtle-page).

  • Image: Connect this plug to a component to provide it with the image of this drawing page (there is only one image pin shown at a time, the one belonging to the top visible page from the stack of pages).

Applet parameters

  • menubar = true | false
    Shows or hides the green bar at the components'
    top. [Default=true    ]
  • PageI = name
    Sets page(I)'s name, where I is a continuous number ³ 1, in the way that follows:
    a. I    f the name assignment is omitted
        (i.e.     PageI = ) then a default name is    
        assigned.
    b. If the name assignment starts with a "_"
        (i.e.     PageI = _name) then the page is to be
        a Turtles' Page with that name.
    c. If the name is just the string "_"
        (i.e.     PageI = _) then the turtle page gets a
        default name.
    All page names must be unique.
  • Image? = image's file path
    Sets Page(I)'s image [optional]
  • Width? = page(I)'s width
    [default=max visible or Image Width] (I same as page number)
  • Height? = page(I)'s height
    [default=Applet/Image Height] (I same as page number)
  • PenColor = RGB color as hex (e.g. FF0000 for red)
    Sets Pen's (foreground's) color [default is black]
  • FillColor = RGB color as hex (e.g. 00FF00 for green)
    Sets the Fill color [default is red]
  • RubberColor = RGB color as hex (e.g. 0000FF for blue)
    Sets Rubber's (background's) color [default is white]
  • FilledShapes = true | false
    Sets the filling color for shape's interior [default=true]
  • Opaque = true | false
    Opaque TurtlePages [default=false]
  • PenSize = pen's size
    Sets Pen's size [default=1]

E-Slate Logo primitives

  • PLOTPIXEL x y
    Plot a point at (x, y) and keep as last position
  • PIXELCOLOR x y
    Returns the color of the point at (x, y) as a list of RGB color elements
  • MOVETO x y
    Move to (x, y) – keep (x, y) as the last position
  • LINE x1 y1 x2 y2
    Draw a line from (x1, y1) to (x2, y2) and keep (x2, y2) as the last position
  • LineTo x2 y2
    Draw a line from the last position to (x2, y2) and keep (x2, y2) as the new last position
  • RECTANGLE x1 y1 x2 y2
    Draw a rectangle with (x1, y1) to (x2, y2) as two opposite corners and keep (x2, y2) as the last position
  • CIRCLE centerX centerY radius
    Draw a circle with (centerX, centerY) as its center and a radius of length radius and keep (x2, y2) as the last position
  • (CIRCLE centerX centerY circumreferenceX circumreferenceY)
    Draw a circle with (centerX, centerY) as its center and with such a radius that the point (    circumreferenceX, circumreferenceY) is on the circle's circumreference. Keeps (centerX, centerY) as the last position
  • OVAL x1 y1 x2 y2
    Draw an oval enclosed in a virtual rectangle with (x1, y1) to (x2, y2) as two opposite corners and keep (x2, y2) as the last position
  • TEXT message x y
    Draw the message string at (x, y) and keep (x, y) as the last position
  • CanonicalPolygon centerX centerY edgeX edgeY edges
    Draw a canonical polygon (Pythagoreian) with its center at (centerX, centerY), one of its edges at (edgeX, edgeY) and a certain number of edges. Keeps (edgeX, edgeY) as the last position
  • SPRAY x y
    Spray at point (x, y) and keep (x, y) as the last position
  • COPY x1 y1 x2 y2
    Copy to the Canvas clipboard the area of a virtual rectangle that has (x1, y1) and (x2, y2) as two of its corners
  • PASTE x y
    Paste to the top visible page with an image from the Canvas clipboard with (x, y) as its top left corner
  • CUT x1 y1 x2 y2
    Cut the area of a virtual rectangle that has (x1, y1) and (x2, y2) as two of its corners and place it in the Canvas clipboard. The cut area gets filled with the current background color
  • CLEAR x1 y1 x2 y2
    Clear with the current background color the area of a virtual rectangle that has (x1, y1) and (x2, y2) as two of its corners
  • STAMP
    Copies the current vector image from a top visible turtle page to the nearest to the turtle page underlying drawing page
  • SetPenSize penSize
    Sets the pen’s size to penSize
  • SetCanvasPageSize width height
    Sets the width and height of the top visible page, stetching any old drawing to this new size
  • CanvasPageWidth
    Returns the width of the top visible page in the Canvas
  • CanvasPageHeight
    Returns the height of the top visible page in the Canvas
  • ClearCanvasPage
    Clears the top visible drawing page by filling it with the background color
  • SetCanvasForeground [red green blue]
    Sets the foreground color’s RGB elements
  • CanvasForeground
    Returns the foreground color as a list of RGB color elements
  • SetCanvasBackground [red green blue]
    Sets the background color’s RGB elements
  • CanvasBackground
    Returns the background color as a list of RGB color elements
  • SetCanvasFill [red green blue]
    Sets the fill color’s RGB elements
  • CanvasFill
    Returns the fill color as a list of RGB color elements
  • CanvasPAINTmode
    Sets the bit-blitting mode to AND (overwrite)
  • CanvasXORmode
    Sets the bit-blitting mode to eXclusive-OR (combine)
